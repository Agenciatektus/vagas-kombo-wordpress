Integre vagas de emprego do Kombo.com.br no seu site WordPress com widget Elementor personalizavel e responsivo.

== Description ==

O plugin **Quadro de Vagas Kombo** permite integrar facilmente as vagas de emprego cadastradas na plataforma Kombo.com.br ao seu site WordPress, utilizando um widget personalizado para o Elementor.

= Recursos =

* **Widget Elementor Completo**: Arraste e solte para adicionar o quadro de vagas em qualquer pagina
* **3 Layouts Disponiveis**: Grid (cards), Lista e Accordion
* **Totalmente Responsivo**: Adaptacao automatica para desktop, tablet e mobile
* **Personalizacao Visual Completa**: Controles de cores, tipografia, espacamento, bordas e estilos
* **Filtros Avancados**: Filtre vagas por localizacao, area, numero de vagas e data de abertura
* **Cache Inteligente**: Sistema de cache configuravel para melhor performance
* **Acessibilidade**: Suporte a navegacao por teclado e leitores de tela (ARIA)
* **Atualizacao Automatica**: Sistema integrado de atualizacoes via GitHub
* **Encoding UTF-8**: Tratamento automatico de caracteres especiais e acentuacao
* **Sem Dependencias Externas**: Codigo leve e otimizado

= Requisitos =

* WordPress 6.0 ou superior
* Elementor 3.0 ou superior
* PHP 7.4 ou superior
* Conta ativa no Kombo.com.br

== Installation ==

1. Faca upload da pasta `quadro-vagas-kombo` para o diretorio `/wp-content/plugins/`
2. Ative o plugin atraves do menu 'Plugins' no WordPress
3. No Elementor, procure pelo widget "Quadro de Vagas Kombo" na categoria "Kombo"
4. Arraste o widget para sua pagina e configure o CID da sua empresa

= Configuracao =

1. Acesse sua conta no Kombo.com.br
2. Va em "Empresa" > "Dados Cadastrais" para encontrar seu Codigo do Cliente (CID)
3. No widget Elementor, insira o CID no campo correspondente
4. Configure as opcoes de exibicao e estilo conforme necessario

== Frequently Asked Questions ==

= Onde encontro o CID da minha empresa? =

O CID (Codigo do Cliente) pode ser encontrado no painel do Kombo:
* Para Kombo Estrategico: Empresa > Dados Cadastrais
* Para Kombo Gratis/Selecao: Ferramentas > Ferramentas Prontas

= O widget nao esta exibindo as vagas =

Verifique se:
1. O CID esta correto
2. Existem vagas ativas e marcadas para exibicao no site
3. A conexao com o servidor Kombo esta funcionando

= Como limpar o cache das vagas? =

O cache e automaticamente renovado conforme a duracao configurada no widget (padrao: 30 minutos). Para limpar manualmente, desative e reative o plugin.

= Posso usar varios widgets na mesma pagina? =

Sim, voce pode adicionar multiplos widgets com diferentes configuracoes ou CIDs na mesma pagina.
